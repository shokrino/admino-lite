Add Persian font, themes, dark mode, and custom styles to your WordPress admin panel and login page — lightweight and easy to use.

== Description ==

After activating the plugin, go to **Settings → Admino Lite** to customize:
- Persian font for admin & login
- Admin themes (Shokrino / Dark Mode)
- Brand color
- Login page background & logo
- White label admin bar & header
- Classic editor for posts & widgets

== Installation ==

1. Upload `admino-lite.zip` to `/wp-content/plugins/` directory.
2. Extract the ZIP file.
3. Activate **Admino Lite** through the 'Plugins' menu in WordPress.
4. Go to plugin settings and customize as you wish.

== Frequently Asked Questions ==

= What is the name of the Persian font? =
It is the free **Shabnam** font.

= Does the dark mode affect the block editor? =
Yes — when dark theme is active, the classic editor & block editor both get dark styling.
