You can easily add Persian font and style to your wp-admin panel.

== Description ==

after activating the plugin you can go to settings and add choose your color and etc.

== Installation ==

1. Upload `admino-lite.zip` to the `/wp-content/plugins/` directory
2. Extract zip file
3. Activate Admino Lite through the 'Plugins' menu in WordPress
